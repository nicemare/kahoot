# Kahoot Auto Get Answer

## Requirements
 * **Python**
      * [Windows installing python](https://www.python.org/downloads/)    
      * [Linux] $ sudo apt-get install python3
 * **GIT**
      * https://git-scm.com/downloads
## How to get started
  ```
  git clone https://github.com/nicemare/kahoot/
  ```
  ```
  cd kahoot
  ```
  ```
  python kahoot.py
  ```
